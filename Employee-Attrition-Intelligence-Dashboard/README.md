Project Title

Employee Attrition Intelligence Dashboard

Objective

Analyze employee attrition patterns using Python and Tableau to identify the key factors influencing employee turnover.

Tools Used:
Python
Pandas
NumPy
Matplotlib
Seaborn
Tableau Public

Dataset:
IBM HR Analytics Employee Attrition Dataset

Project Workflow:
Data Collection
Data Cleaning
Exploratory Data Analysis
Business Insights
Dashboard Development

Key Findings:
Overall Attrition Rate: 16.1%
Sales department has the highest attrition.
Sales Representatives show the highest attrition.
Overtime employees are nearly 3× more likely to leave.
Lower job satisfaction and work-life balance are linked to higher attrition.
